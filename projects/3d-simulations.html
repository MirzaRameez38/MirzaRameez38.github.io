I developed an immersive suite of interactive 3D astrophysical simulations using Unity, enhanced by Generative AI assistance. Both simulations feature intuitive camera controls for seamless navigation, physics-based interactions, and real-time parameter adjustments that instantly update the visualizations. Architecture & Code

These simulations leverage Unity's powerful 3D rendering and physics engines, enhanced by custom shaders and procedural generation techniques. Generative AI assisted in optimizing algorithms and generating realistic visual effects based on scientific data.


void SetupStar()
{
    transform.localScale = Vector3.one * starRadius;
    MeshRenderer renderer = GetComponent();
    renderer.material = neutronStarShaderMaterial;

    if (neutronStarShaderMaterial != null)
    {
        neutronStarShaderMaterial.SetFloat("_StarRadius", starRadius);
        neutronStarShaderMaterial.SetColor("_EmissionColor", TemperatureToColor(surfaceTemperature));
    }
}

// Convert temperature (in Kelvin) to visible color
Color TemperatureToColor(float temperature)
{
    float temp = Mathf.Clamp(temperature, 1000f, 40000f);
    float r = Mathf.Clamp01(Mathf.Pow(temp / 6000f, 4));
    float g = Mathf.Clamp01(Mathf.Pow(temp / 6000f, 2));
    float b = Mathf.Clamp01(temp / 6000f);
    return new Color(r, g, b);
}

Learning Outcomes

This project deepened my expertise in scientific simulation development and AI-assisted design, yielding valuable insights:

Scientific Simulation Development

This project significantly deepened my understanding of translating complex scientific phenomena into interactive visualizations. I learned how to balance scientific accuracy with visual clarity and system performance. Working with astrophysical models required extensive research into neutron star properties and general relativity concepts.

Procedural Generation Techniques

Building these simulations taught me advanced techniques for procedural content generation. Rather than manually creating assets, I implemented systems that generate visual elements algorithmically based on physical parameters. The magnetic field lines, particle effects for radiation jets, and spacetime grid deformations are all generated procedurally, allowing them to adapt dynamically to parameter changes.

Shader Programming for Scientific Visualization

This project pushed me to develop custom shader solutions for representing complex physical phenomena. I learned how to implement vertex displacement in shaders to create the spacetime deformation effect and emission properties to visualize extreme temperatures. This experience gave me practical skills in using Unity's Shader Graph system alongside custom shader code to create effects that would be impossible with standard materials.

Unity Physics Integration

Implementing these simulations required deep integration with Unity's physics system to create realistic interactions. I learned techniques for detecting collisions and translating them into visual effects through shader property updates. The project taught me how to create physics-based controls that feel natural and intuitive to users.
